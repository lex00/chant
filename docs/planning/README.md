# Planned Features

This directory contains design documents for features under consideration. These are not yet implemented but are being explored.

## Contents

- [Git Hooks Automation](./git-hooks.md) - Automated git hook integration
- [Lint Files Flag](./lint-files.md) - `chant lint --files` for targeted validation
- [Activity Reports](./report.md) - `chant report` for team analytics
- [Decisions Frontmatter](./decisions.md) - `decisions:` field for tracking design decisions
- [Spec Templates](./templates.md) - Template system for spec creation

## Status

These features are in the design phase. Implementation details, API design, and acceptance criteria are still being developed.

To propose changes or discuss these features, open an issue on GitHub.
