# v0.13.12

Released: 2026-02-03

## MCP Orchestration Tools

Added 3 new MCP tools for agent orchestration workflows:

### chant_verify
Verify a spec meets its acceptance criteria before finalizing.

```json
{"name": "chant_verify", "arguments": {"id": "001"}}
// Returns: {"verified": false, "criteria": {"total": 5, "checked": 3, "unchecked": 2}, ...}
```

### chant_work_start
Start working on a spec asynchronously (returns immediately).

```json
{"name": "chant_work_start", "arguments": {"id": "001", "chain": true}}
// Returns: {"process_id": "...", "pid": 12345, "mode": "chain", ...}
```

### chant_work_list
List running work processes for monitoring.

```json
{"name": "chant_work_list", "arguments": {"include_completed": true}}
// Returns: {"processes": [...], "summary": {"running": 1, "completed": 5, "failed": 0}}
```

## Enhanced chant_log

Added incremental log polling support:
- `offset` parameter for byte-offset-based reads
- `since` parameter for timestamp-based filtering
- Response includes `byte_offset`, `line_count`, `has_more` for efficient polling

## macOS Code Signing Fix

Fixed SIGKILL (exit 137) on macOS when running chant after installation:
- Release builds now auto-sign macOS binaries
- Homebrew formula includes codesign step
- Documentation updated with troubleshooting guide

## Tool Count

MCP server now exposes **16 tools** (was 13):
- Query tools: 8 (added `chant_verify`)
- Mutating tools: 8 (added `chant_work_start`, `chant_work_list`)
