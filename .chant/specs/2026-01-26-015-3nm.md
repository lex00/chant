# Add interactive wizard mode to chant init command - guide new users through setup

## Overview

When `chant init` is run without any flags, launch an interactive wizard to guide new users through project setup.

## Interface

```bash
chant init                             # Wizard mode
chant init --name myproject            # Direct mode (existing)
chant init --agent claude --minimal    # Direct mode (existing)
```

## Implementation

- Detect if running with no flags (wizard mode) vs with flags (direct mode)
- Use `dialoguer::Input` for project name (with detected default)
- Use `dialoguer::Confirm` for prompt templates (default: yes)
- Use `dialoguer::Confirm` for silent mode (default: no)
- Use `dialoguer::Select` for agent selection with options:
  - None
  - Claude Code
  - Cursor
  - Amazon Q
  - Generic
  - All
- Call existing init logic with gathered values
