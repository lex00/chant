## Example Content

### Template Variables

Prompts support variable substitution using `{{variable}}` syntax:

| Variable | Description | Example Value |
|----------|-------------|---------------|
| `{{project.name}}` | Project name from config | `my-app` |
| `{{spec.id}}` | Full spec ID | `2026-01-24-001-x7m` |
| `{{spec.title}}` | Spec title (from # heading) | `Add user authentication` |
| `{{spec.description}}` | Full spec body | The markdown content |

### Example Prompt

```markdown
# .chant/prompts/standard.md
---
```
