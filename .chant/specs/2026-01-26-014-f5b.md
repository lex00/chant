# Add interactive wizard mode to chant work command - pick from ready specs

## Overview

When `chant work` is run without a spec ID, launch an interactive wizard to select from ready specs.

## Interface

```bash
chant work                             # Wizard mode
chant work 001                         # Direct mode (existing)
chant work --parallel                  # Parallel mode (existing)
```

### Wizard Flow

```
? Select spec to work:
› 2026-01-26-001-abc  Add user authentication
  2026-01-26-002-def  Fix login bug
  2026-01-26-003-ghi  Update API docs
  [Run all ready specs in parallel]

? Select prompt:
› standard
  tdd
  minimal
  documentation

? Create feature branch? No

Working spec 2026-01-26-001-abc with prompt 'standard'...
```

## Implementation

- Load ready specs via existing logic
- Use `dialoguer::Select` to pick spec (show ID + title)
- Include "[Run all ready specs in parallel]" option at end
- Use `dialoguer::Select` for prompt (list available prompts from .chant/prompts/)
- Use `dialoguer::Confirm` for branch mode
- If parallel selected, prompt for max concurrent
