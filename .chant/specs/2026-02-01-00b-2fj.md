# Disambiguate --force flag into specific flags (--force-unlock, --force-retry, etc.)
## Summary

Replaced the ambiguous `--force` flag with specific flags:

- `chant init --force` → `--force-overwrite` (overwrite existing .chant/)
- `chant work --force` → `--skip-deps` + `--skip-criteria` (two separate flags)
- `chant split --force` → `--force-status` (force despite status check)
- `chant archive --force` → `--allow-non-completed` (archive non-completed specs)
- `chant replay --force` → `--skip-validation` (skip validation)
- `chant cancel --force` → `--skip-checks` (skip safety checks)
- `chant delete --force` → `--skip-checks` (skip safety checks)
- `chant site init --force` → `--force-overwrite` (overwrite existing theme files)
