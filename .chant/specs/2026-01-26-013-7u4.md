# Add interactive wizard mode to chant add command

## Overview

When `chant add` is run without a description, launch an interactive wizard to guide spec creation.

## Interface

```bash
chant add                              # Wizard mode
chant add "Fix the bug"                # Direct mode (existing)
```

## Implementation

- Use `dialoguer::Input` for description
- Use `dialoguer::Select` for type
- Use `dialoguer::Input` for labels (parse comma-separated)
- Use `dialoguer::Input` for target_files (parse comma-separated)
- Use `dialoguer::Input` for depends_on (parse comma-separated spec IDs)
- Generate spec with populated frontmatter
