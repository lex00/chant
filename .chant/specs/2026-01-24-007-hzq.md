# Add global config support at ~/.config/chant/

Support global configuration that applies across all projects, loaded from `~/.config/chant/config.md`.

## Acceptance Criteria

- [x] Load global config if it exists

## Implementation Notes

- Use `dirs::config_dir()` or `std::env::var("HOME")` to find home directory
- Global config uses the same format as project config
- Project config values override global config values
- If global config doesn't exist, just use project config
