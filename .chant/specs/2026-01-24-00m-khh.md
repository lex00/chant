# Document --label filter for list command

Add documentation for the `--label` filter flag on the list command (implemented in spec 009-0mq).

## Acceptance Criteria

- [ ] Add `--label` to CLI reference under list command
- [ ] Show example with single label filter

## Example Content

```bash
# Filter by single label
chant list --label auth

# Filter by multiple labels (OR - shows specs matching ANY label)
chant list --label auth --label api

# Combine with --ready
chant list --ready --label feature
```

## Implementation Notes

- Update docs/reference/cli.md in the "Spec Management" section
- The filter uses OR logic: `--label auth --label api` shows specs with auth OR api
