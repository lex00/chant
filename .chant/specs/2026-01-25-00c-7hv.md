## Overview

Add prominent installation instructions showing multiple ways to install chant.

## Installation Methods to Document

### 1. Quick Install (curl)
```bash
curl -fsSL https://github.com/lex00/chant/releases/latest/download/chant-linux-x86_64 -o chant
chmod +x chant
sudo mv chant /usr/local/bin/
```

### 2. Homebrew (macOS/Linux) - if applicable
```bash
brew install lex00/tap/chant
```

### 3. Cargo (from source)
```bash
cargo install --git https://github.com/lex00/chant
```

### 4. Download from Releases
- Link to GitHub Releases page
- List available binaries (Linux, macOS x86_64, macOS aarch64, Windows)

### 5. Build from Source
```bash
git clone https://github.com/lex00/chant
cd chant
cargo build --release
```
