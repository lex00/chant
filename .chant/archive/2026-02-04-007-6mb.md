# Fix model names in init wizard to use short names

## Problem

The init wizard in `src/main.rs` sets model to `claude-sonnet-4`, `claude-opus-4`, `claude-haiku-4` which are invalid model IDs. Claude CLI expects either short names ("sonnet", "opus", "haiku") or full versioned IDs.

## Solution

In `src/main.rs`, change the model selections at lines 1885-1891 and 1981-1987:
- `"claude-opus-4"` → `"opus"`
- `"claude-sonnet-4"` → `"sonnet"`
- `"claude-haiku-4"` → `"haiku"`

Also update tests in `src/config/tests.rs` that use these invalid names.
